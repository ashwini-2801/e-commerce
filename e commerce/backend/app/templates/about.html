Our Story

AuraShop is a fully responsive e-commerce web application engineered for high efficiency, speed, and clean code presentation.

Why AuraShop?

AuraShop showcases modular architecture combining a robust Python Flask backend using object-relational database models and session authorization, and a high-performance frontend styled purely with Vanilla CSS and animated elements.

Administrators have dedicated dashboards with Chart.js analytics integrations, real-time inventory triggers, and direct order tracking status workflows. Customers experience instant updates, secure carts, and search utilities.

Core Stack Details

  • Frontend: HTML5, CSS3 Variables & Grids, Vanilla JS ES6 modules
  • Backend: Python Flask, Blueprints routing structure
  • Database: SQLite local storage with abstraction layers (SQLAlchemy ORM)
  • Security: Session management, Werkzeug password encryption, API validations
